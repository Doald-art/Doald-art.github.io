分类 语法 说明 渲染效果
标题 # H1 到 ###### H6 用1-6个#表示不同级别标题

H1

H6
段落 直接输入文字，段落间空一行 普通文本无需特殊符号 普通文本段落
粗体 **粗体** 或 __粗体__ 用双星号或下划线包裹 粗体
斜体 *斜体* 或 _斜体_ 用单星号或下划线包裹 斜体
删除线 ~~删除线~~ 用双波浪线包裹 删除线
无序列表 - 项目 或 * 项目 符号后加空格 • 项目
有序列表 1. 项目 数字后加句点和空格 1. 项目
任务列表 - [ ] 任务（未完成）- [x] 任务（完成） 需用空格分隔（GFM扩展语法） ☐ 任务☑ 任务
链接 [链接文字](URL)或参考式：[链接文字][id] + [id]: URL 行内链接或分离式定义 Markdown指南
图片 ![替代文本](图片URL)参考式：![替代文本][id] + [id]: 图片URL 语法类似链接，前加! 显示图片
代码块 行内：`代码`块级：用三个反引号包裹（可指定语言） 行内用单反引号，块级用三个反引号 print("Hello")
