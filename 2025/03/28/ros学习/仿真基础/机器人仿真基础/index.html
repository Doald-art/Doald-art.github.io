概念

**机器人系统仿真 **：是通过计算机对实体机器人系统进行模拟的技术，在 ROS 中，仿真实现涉及的内容主要有三:对机器人建模(URDF)、创建仿真环境(Gazebo)以及感知环境(Rviz)等系统性实现。
### 相关组件
1. Urdf:统一的机器人描述格式
2. Rviz:ros 的三维可视化工具
3. Gazebo: 一款 3 D 动态模拟器，用于显示机器人并创建仿真环境。

# Urdf 语法详解
## Link
1. 概念: urdf 中的 Link 标签用于描述机器人的某个部件的外观和物理属性，比如：机器人底座、轮子、激光雷达、摄像头。
2. 作用：在 link 标签内，可以设计该部件的形状、尺寸、颜色、惯性矩阵、碰撞参数等属
3. 案例：分别生成长方形、圆柱与球体的机器人部件

## Joint
1. 概念：urdf 中的 joint 用于描述机器人关节运动学和运动学属性。
2. 属性：
- Name–>为关节命名
- Type –>关节运动形式
- - continuous: 旋转关节，可以绕单轴无限旋转
- revolute: 旋转关节，类似于 continues,但是有旋转角度限制
- prismatic: 滑动关节，沿某一轴线移动的关节，有位置极限
- planer: 平面关节，允许在平面正交方向上平移或旋转
- floating: 浮动关节，允许进行平移、旋转运动
- fixed: 固定关节，不允许运动的特殊关节
3. 子标签
- Parent link (必需的父系坐标系)
- Child link (必需的子系坐标系)

  1. 案例: 创建机器人模型，底盘为长方形，在长方形的前面添加一个摄像头，摄像头可以沿着 z 轴 360 度旋转。

Urdf 实操

  1. 需求描述：
    创建一个四轮圆柱状机器人模型，机器人参数如下,底盘为圆柱状，半径 10cm，高 8cm，四轮由两个驱动轮和两个万向支撑轮组成，两个驱动轮半径为 3.25cm,轮胎宽度1.5cm，两个万向轮为球状，半径 0.75cm，底盘离地间距为 1.5cm(与万向轮直径一致)
  2. 实现流程：
    1. 新建 urdf 以及 launch 文件
  1. 底盘搭建
  2. 添加驱动轮
  3. 添加万向轮
  4. 然后直接运行 launch 文件
